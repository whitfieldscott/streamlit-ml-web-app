# 🏠 Airbnb Price & Insights App (Streamlit)

## 📌 Project Overview
This project is an interactive web application built with Streamlit that analyzes Airbnb listing data in New York City. The app allows users to explore pricing trends and gain insights based on location, room type, and listing characteristics.

The goal is to demonstrate how data analysis and machine learning can be transformed into a user-friendly application for real-world decision-making.

---

## 🚀 Features
- 📊 Explore Airbnb listings by neighborhood and room type
- 💰 View price distributions and trends
- 🔍 Filter data dynamically based on user input
- 📈 Visual insights powered by real dataset
- ⚡ Interactive UI built with Streamlit

---

## 🧠 Methodology

### 1. Data Loading & Cleaning
- Imported dataset from public source
- Cleaned missing values and standardized fields
- Selected relevant features for analysis

### 2. Data Exploration
- Analyzed pricing across neighborhoods
- Compared room types and listing characteristics
- Identified key factors influencing price

### 3. Application Development
- Built interactive UI using Streamlit
- Created filters for user-driven exploration
- Integrated visualizations for insights

---

## 📊 Dataset
- Source: Airbnb NYC dataset
- Includes:
  - Neighborhood
  - Room type
  - Price
  - Minimum nights

## ▶️ How to Run the App

### 1. Install dependencies
    pip install -r requirements.txt

### 2. Run the Streamlit app
    streamlit run src/Streamlit_app.py

### 3. Stop the app
Press `Ctrl + C` in the terminal to stop the application.
